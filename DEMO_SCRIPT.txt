[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png) (pause)
Let's look at the underlying agent architecture. (pause) First, the user's query enters the Security Checkpoint node, which scrubs PII, blocks prompt injections, and restricts payroll access. (pause) Once cleared, it routes to the Orchestrator Agent. The Orchestrator acts as the brain, delegating tasks to two specialist sub-agents: the Document Analyst, which reads text and spreadsheets, and the Risk Assessor, which computes financial numbers. Both sub-agents leverage our custom MCP Server to list files, read documents, and run financial formulas safely. (pause) The Orchestrator then compiles a plan and passes it to the Human Review node, where the system pauses for human approval. (pause) Once the user says 'yes', the Executive Report Agent takes the approved plan and generates a complete, polished business report.
